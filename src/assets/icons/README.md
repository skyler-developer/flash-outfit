# TabBar 图标说明

请在此目录下放置以下图标文件：

## 需要的图标文件

1. **浏览**
   - `browse.png` - 未选中状态的图标（建议尺寸：81px × 81px）
   - `browse-active.png` - 选中状态的图标（建议尺寸：81px × 81px）

2. **创建**
   - `create.png` - 未选中状态的图标（建议尺寸：81px × 81px）
   - `create-active.png` - 选中状态的图标（建议尺寸：81px × 81px）

3. **我的**
   - `mine.png` - 未选中状态的图标（建议尺寸：81px × 81px）
   - `mine-active.png` - 选中状态的图标（建议尺寸：81px × 81px）

## 图标要求

- 格式：PNG
- 尺寸：建议 81px × 81px（小程序标准尺寸）
- 颜色：未选中图标使用灰色，选中图标使用主题色
